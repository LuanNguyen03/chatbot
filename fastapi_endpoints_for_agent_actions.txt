# FastAPI Endpoint Documentation for Importing agent_actions

Below is a summary of all main API endpoints in the project, formatted for easy import into the `agent_actions` table. Each entry includes:
- URL path
- HTTP method
- Short description
- Required parameters (name, type, required/optional)
- Auth type (JWT, none, etc.)
- Timeout & retry (default: 30s, 3 retries if not specified)

---

## Example Format

- **URL:** /api/transfer
- **Method:** POST
- **Description:** Chuyển khoản giữa các tài khoản
- **Params:**
    - from_account: str (required)
    - to_account: str (required)
    - amount: float (required)
- **Auth:** JWT
- **Timeout:** 30s
- **Retry:** 3

---
